
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. It's built with HTML, CSS, and JavaScript, and it uses modern web features like Web Components and CSS variables for a clean and maintainable codebase.

## Current Features

*   **Lottery Number Generation:** Click a button to generate a set of 6 unique random numbers between 1 and 45.
*   **Responsive Design:** The layout adapts to different screen sizes, making it usable on both desktop and mobile devices.
*   **Dark Mode:** A theme toggle allows users to switch between light and dark modes.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The stylesheet for the application.
*   `main.js`: The JavaScript file that contains the application logic.
*   `blueprint.md`: This file, which documents the project.
